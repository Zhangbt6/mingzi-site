# 大模型在控制系统中的应用尝试

用大模型直接控制机器人还不现实，但作为高层任务规划器效果超出预期。

## 方法

将用户指令经过 LLM 解析后输出结构化的行为树（Behavior Tree），再由传统控制器执行叶子节点的动作。

## 实验结果

在仿真机械臂上测试了"把红色积木放到蓝色盒子右边"这类指令，成功率约 70%。失败主要因为：
- 视觉识别把颜色搞混
- 抓取姿态不合理导致中途掉落

## 局限

延迟是最大问题：一次 LLM 调用平均 2-3 秒，还不能用于高速运动控制。

未来方向：模型量化 + 本地部署 + 思维链缓存。
